HALO Link 4.1 发布说明

HALO Link v4.1 – 2025年2月更新

HALO Link 4.1 发布说明

SOFTWARE UPDATE | February 2025
Research Use Only

新功能

  • Figure Maker：创建可发表级别的图像，支持布局/样式控制、6x6 图阵、动态更新、锁定图像框、添加参考框、保存与导出。
  • 整张幻灯片下载：从研究幻灯片网格下载单个图像（限拥有者权限），支持所有受支持的单文件图像格式。
  • HALO Link-Visiopharm 集成：可在 HALO Link 中显示 Oncotopix 生成的标注、ROI 和标签（ROI 与标签作为可交互标记显示；HALO Link 的数据不会在 Visiopharm 中显示）。
  • 视图设置直方图：荧光图像支持通道直方图调整（白入/黑入），可跨通道或单通道调整；预览窗口可实时查看效果；点击 Apply 应用到整张图像；支持通道重命名（Editor+ 权限）。
  • 重新设计的研究导航：新增收藏与最近研究页；可展开的研究树，便于在嵌套研究间高效导航（研究树保持可见）。

标注

  • 标注管理器：在研究内跨幻灯片创建与管理标注层，流转标注工作流；支持批量重命名、改色、显隐与删除。
  • 在 HALO Link 查看器的标注页提供标注层的批量显隐按钮。
  • 标注长度现与面积一起显示，并包含在标注汇总导出中。

作业监控图表（Job Monitor Plot）

  • 在分析（Analysis）页为管理员提供新的作业监控图表。
  • 按日期范围、条件与处理节点筛选分析作业；可选择单个节点用于排障并查看负载。
  • 查看特定时间段内的排队作业总数。
  • 提供作业监控表（Job Monitor Table），展示作业开始/结束时间、分析幻灯片、算法名称与进度百分比等详情。

其他改进

  • 更新分析对话框 UI：更便捷地访问已保存的分析设置与预训练 AI 应用。
  • 标注设置中显示分类器名称，便于识别流水线输出。
  • 新增快捷键快速切换标记：Ctrl + , 与 Ctrl + .
  • 幻灯片查看热图在缩略图上生成叠加，辅助视觉质控（QC）。
  • 新建子研究可继承盲化字段。
  • 日期搜索新增 Between 运算符。
  • 在复制/移动幻灯片与“我的研究”对话框中可直接搜索研究。
  • 分析队列对话框显示唯一的处理节点名称。

移除功能

  • 不再支持远程分类器训练。
  • 不再支持在 HALO Link 中查看分类器训练区域。
